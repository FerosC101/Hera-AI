# HERA Web Dashboard

Next.js/React web dashboard for doctors and administrators.

## Overview
This is the web-based dashboard designed for:
- **Doctors**: View patient data, analyze AI insights, communicate with midwives
- **Admins**: Manage users, monitor system health, access analytics

## Tech Stack
- **Framework**: Next.js
- **Library**: React
- **Language**: TypeScript
- **Styling**: (To be determined - TailwindCSS, Material-UI, etc.)
- **Backend**: Firebase
