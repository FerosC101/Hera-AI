# HERA Documentation

Comprehensive documentation for the HERA project.

## Overview
This folder contains all project documentation including:
- Architecture and design documents
- API documentation
- User guides
- Development guidelines
- Setup and deployment instructions

## Documentation Structure

### `/architecture`
System architecture, diagrams, and design decisions

### `/api`
API specifications and endpoint documentation

### `/guides`
User guides and tutorials for different roles (midwives, doctors, admins)

### `/development`
Development setup, coding standards, and contribution guidelines

### `/deployment`
Deployment guides and infrastructure documentation

## Contributing
Please follow the documentation standards outlined in `/development` when adding new documentation.
