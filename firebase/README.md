# HERA Firebase Backend

Firebase configuration and backend services.

## Overview
This folder contains Firebase configuration, cloud functions, and backend rules for:
- Authentication and user management
- Real-time database/Firestore
- Cloud Storage
- Cloud Functions
- Security rules

## Tech Stack
- **Platform**: Firebase/Google Cloud
- **Services**: 
  - Firebase Authentication
  - Cloud Firestore
  - Firebase Storage
  - Cloud Functions (Node.js/TypeScript)
  - Firebase Hosting
