# HERA AI Services

Python FastAPI backend for AI/ML services and analysis.

## Overview
This service provides AI-powered analytics and insights for maternal health monitoring, including:
- Risk assessment models
- Predictive analytics for complications
- Health metric analysis
- Data processing and insights generation

## Tech Stack
- **Framework**: FastAPI
- **Language**: Python 3.10+
- **ML Libraries**: TensorFlow/PyTorch, scikit-learn, pandas, numpy
- **API Documentation**: Auto-generated via FastAPI (Swagger/OpenAPI)
